# Phone Addiction Analyzer
ML Project analyzing smartphone usage behavior and predicting addiction patterns.
### Features
- User level behavioral analysis from app usage data
- Custom addiction scoring system
- Classification using Logistic Regression
- Insight generation based on productivity and screen time
- Personalised usage recommendations
### Key Insights
- Higher screen time strongly correlates with addictive usage patterns
- Low prodctive app usage increases addiction risk
- Behavioral features improve prediction accuracy
### Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
### Project Structure
- Data preprocessing
- Feature engineering
- Model training
- Evaluation
- Insights & recommendations
### Dataset
- The dataset includes app usage statistics such as screen time, launches, interactions, and productivity labels.
Due to size considerations, the dataset is not included in this repository.
### Outcome- Achieved 95% accuracy in predicting addiction behavior using engineered features.
